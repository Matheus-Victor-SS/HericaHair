## 📌 Sobre o Projeto

O **HéricaHair** é um site institucional desenvolvido para apresentar serviços de transformação capilar, com foco principal em:

* 📱 Experiência mobile premium
* 💬 Conversão direta via WhatsApp
* 📸 Portfólio com efeito "Antes e Depois"
* 🎨 Identidade visual elegante (preto + dourado)
* ⚡ Performance leve (sem frameworks)

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla ES6+)
* Google Fonts (Poppins + Dancing Script)
* Font Awesome 6

---

## ✨ Funcionalidades

### 🧭 Navbar Responsiva

* Menu hambúrguer animado
* Ícone alternável (☰ / ✖)
* Fecha ao clicar fora
* Scroll suave

---

### 🖼️ Galeria Antes e Depois

* Layout em cards
* Hover com efeito zoom
* Lightbox com animação suave
* Fallback automático de imagens

---

### 🎭 Lightbox Interativo

* Ampliação de imagens
* Fechamento por:

  * Clique fora
  * Botão fechar
  * Tecla ESC
* Imagem ilustrativa automática caso falhe o carregamento

---

### 🎬 Animações de Scroll

Implementadas com **IntersectionObserver**:

* Fade-in
* Slide-up
* Transições suaves
* Performance otimizada

---

### 🧠 Sistema Inteligente de Fallback

Caso a logo ou imagens não carreguem:

* Tenta múltiplos caminhos
* Exibe versão alternativa
* Loga no console para debug

---

### 📱 Mobile First

O projeto foi pensado principalmente para:

* Acesso via Instagram
* Clique direto para WhatsApp
* Navegação simples
* Botões grandes e intuitivos
* Experiência fluida no toque
